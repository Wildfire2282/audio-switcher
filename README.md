# Audio Switcher

Windows 系统托盘音频输出设备快速切换与音量管理工具。

![Tray icon](assets/icon.png)

## 功能

- **一键切换输出设备** — 左键托盘图标，菜单顶部列出所有播放设备，点击即可设为默认
- **应用静音控制** — 左键菜单中为每个正在发声的应用提供静音复选框，快速开关单个应用声音
- **全局静音** — 左键菜单顶部可一键静音/取消静音默认设备
- **滚轮调音** — 鼠标悬停托盘图标时滚动滚轮，实时调节默认设备音量，tooltip 显示设备名和百分比
- **开机自启** — 右键菜单中一键启用/禁用 Windows 自动启动
- **设备热插拔** — 插入/拔出音频设备时自动刷新设备列表
- **单实例运行** — 只能运行一个实例，不会重复启动
- **无运行时依赖** — AOT 单文件发布，双击即用

## 使用

1. 从 [Releases](https://github.com/Wildfire2282/audio-switcher/releases) 下载 `audio-switcher.exe`
2. 双击运行（无需管理员权限）
3. 系统托盘出现波形图标后即可操作

## 菜单说明

### 左键菜单

| 项目 | 说明 |
|------|------|
| 输出设备 | 列出所有播放设备，✓ 标记当前默认设备，点击切换 |
| 全局静音 | 勾选/取消勾选以静音或取消静音默认设备 |
| 应用静音 | 列出当前发声的应用，勾选/取消勾选以单独静音 |

### 右键菜单

| 项目 | 说明 |
|------|------|
| 开机自动启动 | 启用/禁用 Windows 开机自启动 |
| 关于 | 显示版本号并打开 GitHub 仓库 |
| 退出 | 关闭程序 |

### 托盘图标操作

| 操作 | 说明 |
|------|------|
| 左键单击 | 打开设备/应用音量菜单 |
| 右键单击 | 打开设置/退出菜单 |
| 滚轮 | 调节默认设备音量 |

## 下载安装

前往 [Releases 页面](https://github.com/Wildfire2282/audio-switcher/releases) 下载最新版 `audio-switcher.exe`，双击运行。
